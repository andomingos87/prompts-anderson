# Slash Command: /plan

Sempre crie planos usando o template oficial em `.context/plans/plan-scafold-document.md`.

## Como aplicar o template
1. Copie o conteúdo completo do template para o novo arquivo de plano em `.context/plans/<nome-do-plano>.md`.
2. Atualize o frontmatter (`status`, `generated`, `agents`, `docs`, `phases`) mantendo a estrutura YAML.
3. Preencha todos os campos `TODO` com informações reais; não remova seções.
4. Mantenha a ordem das seções e o formato das tabelas.
5. Detalhe fases, riscos, dependências, estimativas e rollback de forma objetiva.

## Regra principal
Todos os planos devem seguir esse template sem exceções.
